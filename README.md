Tampilan kalkulator sederhan yang menggunakan bahasa pemrograman html, css dan javascript
Pada kalkulator ini pengguna dapat menginputkan angka dengan menggunakan tombol ataupun keyboard
Kalkulator ini dilengkapi tombol memory untuk menyimpan angka tertentu dan mengoperasikannya
Kalkulator ini responsif dan dapat menangani error seperti pembagian dengan 0 dengan menampilkan "tidak dapat membagi dengan 0"
kalkulator ini dilengkapi juga dengan tampilan riwayat 5 terakhir dari penggunaannya
